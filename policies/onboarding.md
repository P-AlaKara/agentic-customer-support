# Onboarding Policy

## Account Creation
- Customers should use the Sign Up flow with a valid email address.
- Email verification is required before first login.
- Passwords should meet complexity requirements.

## First Login
- Customers should log in with verified email and password.
- If login fails, direct customers to password reset.

## Getting Started
- Encourage profile completion and preference setup.
- Provide first-use guidance for key dashboard features.

## Welcome Tour
- Offer a guided tour for navigation and core tasks.
- Highlight support/help resources available inside the product.
